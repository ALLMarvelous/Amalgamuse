# Amalgamuse

A cosmetic Muse Dash mod allowing you to select a character to appear as without switching abilities.

This mod is still in early development, so some features you might expect (like an in-game UI) are not yet implemented. For now, you can only change your character by editing the `UserData/Amalgamuse.cfg` file.

⚠️ Until a stable release, this mod is not recommended for online play, as there is a chance it could cause incorrect scores or gameplay by activating abilities for your selected visual character. Use at your own risk.

## Does it beat FavGirl?

Yes! Amalgamuse was written from the ground up to be much more simplistic and efficient than FavGirl, and solves a handful of fundamental issues with it, including:

- Incompatibility with Neon Egg, Rin & Len, and Racer Rin
- Incorrect victory screen for custom charts
- Requiring a manual update for every new character release
- A lot of bloat and unmaintainable code

It also lets you select from all four of Neon Egg's appearances!
